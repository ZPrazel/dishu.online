🥊 站架与步法

基本站架（正架 Orthodox）

  • 脚：左脚前右脚后，间距约肩宽；后脚跟微抬，重心偏前脚掌
  • 膝：微屈，弹性状态——"像坐在高脚凳边缘"
  • 髋：微微内收，保护腹部
  • 上身：略侧对对手，减少正面暴露面积约 30%
  • 手：前手探距（下巴高度），后手贴脸颊护下巴
  • 肘：紧贴肋骨——护肝护脾
  • 下巴：内收，藏于前肩后方
  • 视线：看对方胸部/锁骨——余光感知全身，不盯着拳看

步法体系

步法执行用途训练方法
滑步（Step）移动脚先动，后脚跟；双脚永不交叉基础前后左右位移镜前慢速滑步 3min×3
跳步（Bounce）小幅弹跳保持节奏，脚掌着地节奏控制、试探距敌跳绳 3min×3，感受脚掌弹性
转步（Pivot）前脚为轴，后脚弧线旋转 45-90°创造角度、逃离围绳绕锥桶 8 字转步
突进步（Lunge）后脚猛蹬，前脚大步跨入攻击距离快速缩短距离从绳圈外突进—出拳—退出
侧闪步（L-Step）向斜前/斜后 45° 跨步创造反击角度绳梯训练 + L 步出拳
撤退步（Pull Back）前脚推地、上身微仰、整体后移紧急脱离沙袋摆荡练习进退出入

💡 步法黄金法则

「移动脚先动，重心不丢。」初学者最常见的问题：上身先动、脚跟不上。让步法成为肌肉记忆的唯一方法是——每次训练专门留 5-10 分钟练步法，哪怕只是在镜子前走。

📏 距离控制

拳击里有一句话：「控制了距离，就控制了比赛。」这是区分新手和好手的核心能力。

三层距离模型

距离描述你的武器对手能做什么
远距离
Long Range		 双方前手勉强能碰到或碰不到 刺拳探距、假动作、观察 也只能刺拳
中距离
Mid Range		 双方直拳和部分勾拳能命中 直拳、勾拳、组合 直拳、勾拳
近距离
Inside / Pocket		 贴得很近，双方肘可触身 短勾、上勾、身体拳 同上 + 可能箍颈/搂抱

距离控制的四项能力

  • 感知距离 — 不需要看脚，凭"感觉"知道自己在什么位置。通过大量空击和沙袋进出训练获得
  • 进入距离 — 用刺拳掩护、假动作、角度步法安全地从中远距离切到中近距离
  • 保持在有利距离 — 如果你臂展长，钉在远距离用刺拳折磨对手；如果你力量大，想法进入中近
  • 脱离距离 — 打完组合立刻退出，不被反击。退出的步法和进入同样重要

    • 🔑 一个核心练习：进出 drill

    沙袋前画出三道线（远-中-近）。进 → 打组合 → 出，每组练 2 分钟。重点不是拳，是进的时机和出的速度。这个练 100 次，实战时你会感谢自己。

    刺拳：你的距离雷达

    刺拳不只是轻拳，它是你的测距仪。不确定距离？出刺拳。对手移动？出刺拳。想进？先刺拳再进。把刺拳当作你在黑暗中伸出的手——它告诉你一切。

    👊 拳法体系

    六种基础拳法

    #拳法核心要点发力链最常出错
    1 刺拳 Jab 快、准、回收快。不是重拳，是"雷达拳" 前脚蹬→髋微转→肩送→臂展→拳峰着→原路收回 肘外翻、下巴上抬、不收拳
    2 后直 Cross 全身发力，旋转是关键。收拳和出拳同样重要 后脚蹬地→转髋→转肩→臂推直→拳峰着→原路收回 只用手臂力、前脚离地、重心前倾过多
    3 前勾 Lead Hook 肘抬至肩高、拳心向内、走最短弧线 前脚旋转→转髋→肩带手臂弧线击出 挥太大像棒球、肘低于肩变摆拳
    4 后勾 Rear Hook 力道最大的弧线拳，常用在中距离和反击 后脚蹬→髋猛转→肩带动后手弧线击出 同上 + 重心过度前倾
    5 前上勾 Lead Uppercut 膝微屈→身体沉→手臂贴身向上钻出 微蹲→蹬腿→转髋→手臂沿身体中线上升 手先动身体不沉、距离太远打空
    6 后上勾 Rear Uppercut 近身杀器。必须沉身后再爆发上升 深蹲→猛蹬→转髋→手臂贴身钻升 同前上勾 + 幅度过大露空档

    变体与特殊拳法

    • 身体刺拳（Body Jab）：刺拳下沉打腹部，迫使对手手位下移，为上面开路
    • 拍击刺拳（Pawing Jab）：不全力收回，连续轻拍对手视线（Lomachenko 常用）
    • 过顶拳（Overhand）：后手弧形越过对手前手，打太阳穴/耳后，破高护防守
    • 铲勾（Shovel Hook）：45° 弧线击打肝区/脾区。井上尚弥和 Canelo 的招牌
    • 直摆变线：直拳出到中途突然走弧，破防守节奏

    💡 出拳时呼气

    每拳配合短促的"嘶——"声呼气。不憋气 = 不累。李小龙的嘶声不是作秀，是科学：呼气释放核心压力、保持放松、延长体力。现在就开始习惯它。

    🛡️ 防守体系

    防守金字塔 — 从最优到最次

  • 距离控制（最优）— 让对手根本打不到你
  • 头部移动 — 闪躲、侧闪、下潜、后撤
  • 格挡/拍挡 — 手套、手臂护住要害；或拨开来拳
  • 硬扛（最次）— 收紧肌肉硬接，不得已而为之
    • 技术执行要点防什么
    侧闪 Slip上身向斜前小幅侧移，头从拳路线偏开；闪完立刻回到防守位直拳
    下潜 Roll/Duck膝弯曲→上身弧形下沉→从勾拳下方滚过→回到站架勾拳、摆拳
    后撤 Pull上身微仰，前脚可小步后撤，保持平衡直拳连击
    拍挡 Parry手掌/前臂小幅度拨开拳面——"拍"不是"推"刺拳、直拳
    高护 High Guard双拳贴太阳穴，肘收护肋，透过拳缝看对手组合拳轰炸
    肩滚 Shoulder Roll前肩抬起挡拳，后手护另一边脸，上身微转泄力直拳（高级）
    阻截反击 Catch & Shoot后手手套接住刺拳同时前手直接反击刺拳

    防守的终极心法

    防守不是为了不挨打，是为了打完反击。 每一次闪躲、每一次拍挡，紧接着应该有一个反击。纯粹的防守最终一定会被突破。防守→反击，把这个连起来练。

    ⚡ 组合拳库

    Level 1 — 双拳组合（入门）

    1-2 刺拳 → 后直 万物之母，练到不需要想
    1-1 双刺 刺拳→刺拳，第二拳加力或变角度
    1-3 刺拳 → 前勾 直→弧，节奏切换
    2-3 后直 → 前勾 交叉连击，后手打完前手已在路上
    2-5 后直 → 前上勾 直拳破防→上勾钻入

    Level 2 — 三拳组合（核心）

    1-2-3 刺拳 → 后直 → 前勾 经典三连，打沙袋标配
    1-2-5 刺拳 → 后直 → 前上勾 上头→上头→下巴
    1-3-2 刺拳 → 前勾 → 后直 节奏变化：快-快-快 / 快-慢-快 都行
    2-3-2 后直 → 前勾 → 后直 后手起→前勾衔接→后手收尾
    1-2-1 刺拳 → 后直 → 再刺拳 进攻后掩护撤退

    Level 3 — 四拳及以上（进阶）

    1-2-3-2 刺拳 → 后直 → 前勾 → 后直
    1-1-2-3 双刺 → 后直 → 前勾 双刺设伏，后面才是真货
    1-2-5-2 刺拳 → 后直 → 前上勾 → 后直
    1-2-3-6 刺拳 → 后直 → 前勾 → 后上勾 头部→身体切换
    1-2-1-2 刺拳 → 后直 → 刺拳 → 后直 快-慢-快-慢，节奏欺骗

    Level 4 — 身体打击组合

    1-2b-3 刺拳(头) → 后直(身) → 前勾(头) 高→低→高，经典欺骗链
    1-3b-2 刺拳(头) → 前勾(肝) → 后直(头)
    2b-3b-6 后直(身) → 前勾(身) → 后上勾(头) 全打身体后炸上头
    1-2-3b-2 刺拳 → 后直 → 前勾(肝) → 后直 井上尚弥招牌连击

    Level 5 — 防守反击组合

    Slip → 2-3 侧闪 → 后直 → 前勾 闪开刺拳后反击
    Parry → 2 拍挡 → 后直 拍掉刺拳直接后直
    Duck → 5-2 下潜 → 前上勾 → 后直 躲勾拳后反击
    Pull → 1-2 后撤 → 刺拳 → 后直 后撤创造距离后反攻

    💡 组合训练原则

    不要在沙袋上随机打拳。每回合只练 1-2 个组合，重复 50-80 次。先慢后快，先把流畅度搞定，速度是最后才加的东西。一个流畅的慢组合好过十个乱七八糟的快组合。

    🎵 节奏与时机

    什么是节奏

    节奏 = 出拳的时间间隔模式。人类本能会适应固定节奏。打破节奏 = 创造机会。

    节奏类型例子效果
    匀速1…2…1…2…容易被预判，但可作为"设伏"用
    加速1..2..12..1212!逐渐加速让对手跟不上
    断奏1……1-2!…………1长时间静默后突然爆发，最难防
    切分1-2…3…………2!在"不该出拳的拍子"出拳

    时机是什么

    时机 = 在对手最脆弱的那一瞬间出手。几个黄金时机窗口：

    • 对手出拳时 — 他伸拳 = 他那一侧暴露
    • 对手收拳时 — 收回来的路上，他的防守还没归位
    • 对手呼吸间隙 — 刚组合打完有个换气停顿
    • 对手步法落地瞬间 — 脚刚落地那一刻重心最稳也最僵
    • 对手眨眼/视线转移 — 眼睛不看你的那一瞬间

    🔑 发展时机感的练习

    • 双人节奏 drill： 一人出刺拳，你练习在他刺拳收回时出刺拳
    • 沙袋摆荡： 推沙袋让它荡回来，在沙袋向你靠近时出拳（模拟对手进入）
    • 跳绳变速： 30秒慢 → 30秒快 → 30秒双摇，训练节奏切换本能
    • 看比赛时"点节奏"： 自己用手指敲桌子跟上拳手的出拳节奏

    🪞 空击套路

    职业拳手每天空击可能比打沙袋还多。空击不是"随便比划"，是有目的的技术打磨。

    🔹 短版热身（5min）

    1. 1min：慢速基础拳法逐一检查（Jab → Cross → Hook → Uppercut），纠动作
    2. 1min：步法移动 + 刺拳（前后左右绕圈，保持站架）
    3. 1min：1-2 组合 + 头部移动，每打完立刻移动
    4. 1min：自由组合，加防守动作
    5. 1min：全速输出（模拟最后一回合）

    🔹 标准版（12min：4R×3min）

    1. R1 — 技术：慢动作只打1-2和1-2-3。关注每次转髋、回收、下巴位置
    2. R2 — 步法：移动中出拳。重点是出拳瞬间脚下稳定，打完立刻挪
    3. R3 — 防反：模拟对手来拳→防守→反击（Slip-2-3 / Parry-2 / Duck-5-2）
    4. R4 — 自由：全速混合所有技术，每 30 秒换一种节奏

    🔹 长版专项（20min）

    标准版 12min + 以下两项：

    • R5（4min）：身体打击专项 — 每一拳后接一个身体拳，练高低切换
    • R6（4min）：条件限制 — 只用前手 / 只能防不能还 / 只能在"绳边"打

    🎭 拳击风格流派

    了解不同风格，有助于你找到自己的方向，也方便在实战中识别对手类型。

    🥊 外线拳手 Out-Boxer

    代表：Floyd Mayweather, Larry Holmes, Oleksandr Usyk

    利用步伐和刺拳在远距离控制比赛。打点得分、极少受伤。核心能力：步法 + 刺拳 + 距离感。

    💣 重炮手 Slugger / Puncher

    代表：Deontay Wilder, Julian Jackson, George Foreman

    一拳定胜负。不追求点数，追求 KO。步法可能粗糙但力量恐怖。对体能和时机要求极高——因为重拳很耗体力。

    🔥 压迫型 Swarmer / Pressure

    代表：Julio César Chávez, Joe Frazier, Gennady Golovkin

    不断前压、高频出拳、身体打击凶狠。让对方无法喘息。需要超强体能和"能吃拳"的下巴。

    🎯 全能型 Boxer-Puncher

    代表：Manny Pacquiao, Canelo Alvarez, Naoya Inoue

    兼具外线技术和重拳能力，能根据对手切换打法。最全面但也最难——你需要什么都会。

    🥋 反击型 Counter-Puncher

    代表：Juan Manuel Márquez, Canelo, James Toney

    等对手先出手，利用防守+反击得分。极其依赖时机感和预判。看起来"被动"，但反击往往最致命。

    🇨🇺 古巴/苏联体系

    代表：Vasiliy Lomachenko, Guillermo Rigondeaux, Bivol

    业余拳击背景深厚，步法极其灵动，经常切换正反架。强调角度创造和"打了就跑"的移动。

    💡 你现在不需要"选风格"

    20 节课的阶段，你的风格还没成型——这反而是好事。什么都试一试，让教练帮你发现你的身体条件（臂展、速度、力量）更适配什么风格。风格是练出来的，不是选出来的。

    🔧 常见错误自查

    错误表现怎么改
    下巴上抬出拳时下巴离开胸口空击时下巴夹网球练习；或想象下巴压着一个鸡蛋
    不收拳拳打出去后停在前面或掉下来出拳后原路收回防守位；沙袋训练强制回收
    刺拳肘外翻刺拳时肘向外张开变"推"面对墙壁练刺拳，肘擦墙走
    后直不转脚只用上半身，拳无力慢动作刻意练后脚→髋→肩的传导链
    勾拳幅度过大手臂甩太开像挥棒球棒贴墙练勾拳，肘贴身走最短弧线
    步法交叉双脚交叉，重心瞬间不稳绳梯训练，永远移动脚先动
    憋气出拳出拳时屏息，两拳后就没体力每次出拳配合短促"嘶"声呼气
    盯着对方拳看视线追拳，反应总慢半拍练习看胸部/锁骨，用余光感知来拳
    重心过高膝盖太直像站着打空击时想象坐高脚凳，始终保持微屈膝
    组合间停顿过长打一拳看一下反应再打用节拍器或音乐练组合连续性

    🔥 热身流程

    别再"随便跳两下"了。一套标准化的热身让你训练质量提升 30%。

    标准热身（10-12min）

    1. 全身激活（2min）：开合跳 30s + 高抬腿 30s + 原地小跑 1min
    2. 关节活动（3min）：颈→肩→腕→髋→膝→踝，每个关节 30s 绕环。重点是肩和髋
    3. 动态拉伸（3min）：弓步转体、腿摆动（前后+侧向）、猫牛式、肩胛激活
    4. 跳绳（3min）：激活脚掌弹性和全身协调。可以 1min 慢 + 1min 快 + 1min 自由
    5. 空击轻量（1min）：慢速出拳，检查今天身体状态

    🚫 热身禁忌

    • 不要静态拉伸（压腿、拉筋）→ 练完再做
    • 不要一上来就全力 → 身体还没准备好
    • 不要跳过热身 → 肩关节伤病很多跟热身不足有关

    💪 体能专项

    拳击体能的关键理解

    拳击不是马拉松，是间歇性爆发运动：3 分钟高强度 + 1 分钟休息，循环。你的体能训练应该模仿这个模式，而不是去跑 10 公里。

    必练项目

    项目为什么重要怎么练
    跳绳脚掌弹性、节奏感、协调性3R×3min，间歇 1min；逐步加回合
    波比跳全身爆发 + 心肺模拟实战强度20s 全力 + 10s 休息 × 8 组（Tabata）
    核心训练所有拳法力量都通过核心传递平板、俄罗斯转体、死虫式、悬挂举腿
    颈部强化抗震荡能力。职业拳手的必修课颈部桥（谨慎）、弹力带颈部对抗、点头/转头阻力训练
    腿部力量步法移动、发力根基深蹲、弓步、保加利亚分腿蹲
    肩部耐力手举着护下巴 3 分钟比你想的累持轻哑铃（1-2kg）保持防守姿势 2-3min

    💡 体能训练的黄金法则

    练拳击体能≠练健美。你想要的是功能性力量——爆发力、耐力、旋转力量——不是大块肌肉。自重训练 + 弹力带 + 跳绳 > 卧推 + 二头弯举。每次力量训练后，加一组"疲劳空击"（练完力量立刻打 2 分钟空击），这才是拳击体能。

    🍎 营养与恢复

    练前（1-2小时）

    • 碳水为主：香蕉、燕麦、全麦面包
    • 适量蛋白：鸡蛋、酸奶
    • 避免大量脂肪（消化慢、影响训练状态）

    练后（30-60min 黄金窗口）

    • 蛋白质 20-30g：鸡胸、鱼肉、蛋白粉
    • 碳水补充：红薯、米饭、水果
    • 补水至少 500-750ml

    恢复金字塔

  • 睡眠（基石） — 7-9 小时。比任何营养补剂都重要。深睡时生长激素分泌、身体修复
  • 营养 — 蛋白质 + 碳水 + 水。缺一不可
  • 主动恢复 — 散步、轻度游泳、瑜伽。促进血液循环但不增加疲劳
  • 物理手段 — 泡沫轴（每块肌肉 30-60s）、冰敷（关节/软组织急性酸痛时 15min）

    • ⚠️ 伤病预警信号

    • 手腕/拳峰持续疼痛 → 检查缠手和拳面是否正确
    • 肩关节弹响+疼痛 → 可能肩袖问题，减少大幅度摆拳
    • 头晕/恶心/视力模糊 → 可能脑震荡迹象，立即停，就医
    • 肋骨刺痛 → 可能骨裂，别硬扛
    • 拇指扭伤 → 常见于勾拳握拳错误

    原则：关节痛 = 停。肌肉酸 = 可以继续。拿不准就休息，休息一天不会退步，带伤训练会倒退一个月。

    🧤 装备指南

    装备推荐说明
    拳套 沙袋用 12-14oz；实战用 16oz 入门品牌：Fairtex、Twins、Venum、Title。oz 越大缓冲越好。别光图便宜——保护你的手
    缠手带 墨西哥式（弹力）180 英寸 保护拳峰 + 手腕稳定。学一种缠法并每次都一致。比拳套本身更重要
    护齿 定制最佳，其次 boil-and-bite 任何实战必须戴。为上排牙定制，确保呼吸通畅
    头盔 Full-face 或 Open-face 实战必备。不防脑震荡（别误会），但防割伤/擦伤/眉骨裂
    护裆 拳击专用护裆 硬实战和比赛必戴。训练中意外时有发生
    拳击鞋 高帮拳击鞋 等训练稳定后再买。提供脚踝支撑和 pivot 灵活性。比跑鞋安全得多
    跳绳 钢丝速度绳或皮质拳击绳 PVC 绳入门即可。别用太轻的——你需要在手腕上感受到它

    💡 装备优先级

    缠手带 > 拳套 > 护齿 > 头盔 > 护裆 > 拳击鞋。 好的缠手带和拳套是对自己手的投资。手伤了 = 什么都练不了。

    ❓ 问教练的好问题

    很多学员花大钱上私教却不知道问什么。以下问题能让每节课效率翻倍：

    技术类

    • "我刚才那个刺拳，回收的速度对吗？哪里可以更快？"
    • "我的后直转髋到位了吗？帮我看看我的脚。"
    • "这个组合我哪里最不流畅？你帮我慢动作拆一遍。"
    • "我在什么距离最适合出这个组合？远了会怎样？"

    理解类（帮你建立拳击思维）

    • "如果对手一直用刺拳点我，我有什么选择？"
    • "你觉得我现在最大的短板是什么？下个月该重点练什么？"
    • "跟我体型差不多的拳手，有哪些值得我学习？"
    • "刚才实战里那个情况，我应该怎么处理？"

    实用类

    • "这周我自己训练的时候，最该练什么？给一个重点就行。"
    • "帮我录一段我打沙袋的视频，对比着给我讲。"

    💡 一个好习惯

    每节课前想好 一个 要问的问题。不用多，一个就够了。带着问题去上课，比空着脑子去强十倍。

    ⚖️ 街头自卫：法律边界

    ⚠️ 核心原则

    你有拳击训练经历，在司法实践中可能被视为「具有特殊格斗技能」，一旦动手法律上对你不利。拳击技术是最后防线，不是第一选择。

    中国大陆正当防卫框架

    • 《刑法》第20条：为使本人或他人的人身、财产等免受正在进行的不法侵害，采取制止行为 → 正当防卫，不负刑事责任
    • 防卫过当：明显超过必要限度造成重大损害 → 应负刑责但减轻或免除
    • 特殊防卫：对行凶、杀人、抢劫、强奸、绑架等严重危及人身安全的暴力犯罪 → 造成伤亡不负刑责

    ⚡ 四条生死线

    • 对方停了你必须停 — 追击 = 故意伤害，不是防卫
    • 对方空手你用武器 — 可能被认定为防卫过当
    • 口角挑衅后动手 — 不算"不法侵害正在发生" → 互殴
    • 能跑不跑 — 法律鼓励逃避而非对抗。跑了再报警是最优解

    武力使用决策链

  • 能跑吗？ → 跑（零法律风险，最优选择）
  • 能降级吗？ → 语言缓和、道歉（哪怕不是你的错）、离开
  • 只是推搡？ → 继续降级或离开，不要升级成拳脚
  • 持械 / 多人 / 威胁生命？ → 这才是可以动手的底线

    • 🗣️ 冲突降级

    语言降级话术（学会这些，比后手直拳更有用）

    • "兄弟，不好意思，我不该……（承认"错误"，给对方台阶）"
    • "你说得对，我这就走。"
    • "没必要闹大，警察来了对谁都不好。"（第三方威慑）
    • "我家里人在等，先走了。"（示弱 + 合理离开理由）

    非语言降级

    • 手掌摊开前伸 — "我没有威胁"的通用信号
    • 保持一臂以上距离
    • 避免长时间直视 — 在某些文化中是挑衅。看鼻梁或胸口
    • 侧身站立 — 减小正面面积，同时方便移动
    • 动作缓慢 — 任何突然动作都可能被解读为攻击

    🛡️ 街头实战技术

    ⚠️ 街头 ≠ 擂台

    • 没有拳套 → 你的手骨远比对方头骨脆弱
    • 没有裁判 → 倒地后可能被踢头
    • 没有回合 → 10 秒内决定胜负
    • 没有规则 → 对方可能有刀、有朋友
    • 有监控/目击者 → 你的一切行为会被法庭逐帧审查

    拳击技术的街头适配

    技术擂台用法街头调整
    站架拳套护脸、明显格斗姿态手放低更自然，掌心可朝前——既是"别过来"信号也是战斗姿态
    刺拳拳峰击打面部改为掌底推击（Palm Strike）→ 保护手骨，法律上也更像"推开"
    直拳打头部打身体：腹部、胸骨、软肋。打脸极易伤手（拳峰 vs 牙齿=你输）
    勾拳中距离弧线不推荐 → 容易被抓/抱摔。只在贴身缠斗中考虑短勾打身体
    上勾拳近身打下巴身体上勾是最优拳法：打腹部/胃部→瞬间丧失战斗意志、不伤手
    防守高护/拍挡/闪躲改为控距+护头：前手探出保持距离，后手护下巴

    打击目标优先级（街头版）

    优先级目标效果你的风险
    🥇 第一膝盖侧向踢击立即丧失移动能力低
    🥈 第二腹部/胃部重击剧痛、呼吸困难、战斗意志崩溃低
    🥉 第三肝区（右肋下）剧痛、瞬间无力、跪地低
    第四下巴侧面可能 KO中（伤手）
    第五鼻子大量流泪、视线模糊中
    最后面部/头部威慑高（对你和对方法律上都是）

    ⚡ 记住两件事

    ① 身体 > 头部。让对方痛到停手，比把对方打晕在法律上安全得多。

    ② 打完就跑。一旦对方失去攻击能力，立刻撤离，报警备案（抢占先机，证明你是防卫方）。

    🧠 心态建设

    训练心态

    • 过程 > 结果 — 关注"今天比昨天好一点"，而非"什么时候能打实战"
    • 慢就是快 — 慢动作打磨是进步最快的方式。快但错误 = 练错
    • 接受平台期 — 进步不是线性的。两周没感觉？正常。突破在拐角
    • 空击 ≠ 浪费时间 — 顶尖拳手每天空击可能比打沙袋还多

    实战心态

    • 恐惧正常 — 每个拳手都紧张。恐惧让你敏锐，不是软弱
    • 呼吸是第一技术 — 紧张→屏息→快速疲劳。主动慢慢吐气
    • 被打不可怕 — 越怕被打身体越僵，越容易被打。放松才有速度
    • 不确定就出刺拳 — 刺拳是你的"雷达"，帮你搞清楚状况

    💡 拳击是下棋

    不是比谁更能扛，是比谁更能读到对方。每一拳都是信息（刺拳探距、假动作看反应），根据信息做决策。当你在"想"着打而不是"蛮"着打时，已经超越了 80% 的练习者。

    级别强度目的大约何时开始

    • 永远戴护齿 + 头盔（硬实战加护裆）
    • 教练必须在场

    • 实战后不做重体能（脑部需要恢复）
    • "轻一点"必须说出口——保护你的大脑，别不好意思
