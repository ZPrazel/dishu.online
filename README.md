# 地书 · 个人知识操作系统

地书是一个自用优先、持续生长的个人知识操作系统。它不是单纯资料仓库，而是把知识整理成路线、清单、工具箱和判断框架，方便长期学习、实践和复盘。

## 当前结构

- `index.html` — 实用型知识门户
- `kb/` — 9 个知识领域文章
- `assets/css/site.css` — 全站公共样式
- `assets/js/site.js` — 全站公共交互
- `CNAME` — 绑定 `dishu.online`

## 设计方向

- 先服务自己：高频可用、容易打开、容易继续扩展。
- 保持静态站：不引入后端、不依赖构建工具，适合 GitHub Pages。
- 面向未来知识产品：优先沉淀路线图、检查清单、工具箱和实践框架。

## 本地预览

直接在浏览器打开 `index.html` 即可，无需服务器。
